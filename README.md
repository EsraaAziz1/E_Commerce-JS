# E-commerce JS

A simple E-commerce web application built with plain JavaScript, HTML, and CSS.  
Features product listing, shopping cart, and local storage management.  
Perfect for practicing core JavaScript concepts and building interactive web apps from scratch.

## Features
- Display products dynamically
- Add/remove items from shopping cart
- Persist cart data using Local Storage
- Responsive layout

## Technologies Used
- JavaScript 
- HTML4
- CSS2
